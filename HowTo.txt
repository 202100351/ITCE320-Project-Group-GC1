1- Make sure to have the server and client scripts in the same directory
2- You HAVE TO install "prettytable" and "requests" libraries
3- (Make sure the port number (49994) is available for server socket to bind).
4- run the server script first and put ICAO code for a choosen airport
5- wait until the server display
"Server is ready to accept connections from clients"
6- run client script (one or multiple) and type client name
7- request options for the client from (a to d) and (e) to quit
8- At the end, you have to manually terminate the server
